

*The data set defined as Return_DB.csv contains the historical returns:

-The second column is the benchmark.
-The asset universe consists of the columns from the third to the last.

*To run the algorithm, please run the file main_algo.m.

*To check the in-sample graphics, run the file in_sample.m.
